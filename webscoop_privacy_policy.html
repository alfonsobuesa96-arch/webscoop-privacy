WebScoop

Privacy Policy · Last updated: June 13, 2026

WebScoop ("the Extension") is a Chrome browser extension that extracts web content from pages you visit and saves it locally as a ZIP file. This policy explains how the Extension handles your data.

1. Data We Collect

WebScoop does not collect, transmit, or store any personal data on external servers. All processing happens locally inside your browser.

The Extension reads the content of web pages you actively choose to extract (text, images, links, metadata, cookies, etc.) solely to produce the ZIP file you download. This content never leaves your device.

2. Data Storage

The Extension stores the following data locally in your browser using chrome.storage.local:

  • Extraction history — a list of the last 50 URLs you extracted (title, domain, date). Stored only on your device and never synced or shared.
  • User preferences — your selected extraction options (which content types to include). Stored only on your device.

You can clear this data at any time from the Extension's History tab.

3. Permissions Used

WebScoop requests the following Chrome permissions and uses them solely to perform the extraction you request:

  • activeTab / tabs — to read the content of the current tab when you click Extract.
  • scripting — to inject content scripts that extract page data.
  • cookies — to export the cookies of the page you are extracting, if you enable that option.
  • downloads — to save the generated ZIP file to your Downloads folder.
  • storage — to save your preferences and history locally.
  • contextMenus — to add the "Extract with WebScoop" option to the right-click menu.
  • notifications — to notify you when an extraction is complete.
  • clipboardWrite — to copy Markdown to your clipboard when you use the Copy Markdown feature.
  • host_permissions (<all_urls>) — required to extract content from any webpage you visit.

4. Third-Party Services

WebScoop does not use any third-party analytics, tracking, advertising, or telemetry services. No data is sent to any external server.

The Extension uses the following open-source libraries, which run entirely in your browser:

  • Mozilla Readability.js — for clean article extraction.
  • JSZip — for ZIP file creation.

5. Data Sharing

We do not sell, trade, rent, or share any user data with any third party. There is no user data to share.

6. Children's Privacy

WebScoop is not directed at children under 13. We do not knowingly collect any information from children.

7. Changes to This Policy

If we update this policy, we will update the "Last updated" date above. Continued use of the Extension after changes constitutes acceptance of the updated policy.
